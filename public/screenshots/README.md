# Screenshot Files

Place your Mind Clear app screenshots in this directory with the following filenames:

1. `brain-dump.png` - Guided Brain Dump screen
2. `refine.png` - Refine Your Tasks screen
3. `prioritize.png` - Prioritize Tasks screen
4. `ai-chat.png` - AI Chat Assistant screen
5. `tasks.png` - Task Management screen
6. `calendar.png` - Calendar Month View screen
7. `calendar-day.png` - Calendar Day View screen
8. `goals.png` - Goal Tracking screen
9. `profile.png` - Profile & Settings screen

## Image Requirements

- Format: PNG (recommended) or JPG
- Aspect Ratio: 9:16 (portrait, mobile phone screenshots)
- Recommended size: 1080x1920px or similar high resolution
- The images will be automatically optimized by Next.js

## Notes

The screenshots will be displayed in a beautiful carousel gallery on the main page. You can navigate through them using:
- Arrow buttons
- Thumbnail grid at the bottom
- Keyboard arrows (when in fullscreen mode)
- Click any thumbnail to jump to that screenshot
